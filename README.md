# Data Professional Survey Dashboard

## Overview

This Power BI project explores survey responses collected from data professionals around the world. The dashboard provides interactive insights into salary trends, job roles, programming language preferences, employee satisfaction, and career entry challenges within the data industry.

The project demonstrates end-to-end business intelligence development, including data transformation, modeling, visualization, and KPI reporting using Power BI.

## Business Problem

The data industry continues to expand rapidly, creating opportunities for professionals entering analytics, engineering, and data science roles. However, understanding career trends, compensation expectations, job satisfaction, and industry preferences can be challenging.

This dashboard answers key questions such as:

- Which data roles earn the highest salaries?
- What programming languages are most popular among professionals?
- How satisfied are data professionals with their salaries and work-life balance?
- How difficult is it to break into the data field?
- What countries contribute the largest share of survey responses?

## Dashboard Features

### Country Distribution

A treemap visual displays survey participation by country, helping identify geographic representation across the dataset.

### Salary Analysis

Compare average salaries across roles including:

- Data Scientist
- Data Engineer
- Data Architect
- Data Analyst
- Database Developer
- Students and Career Changers

### Programming Language Preferences

Analyze the most commonly used programming languages among data professionals.

Languages included:

- Python
- R
- SQL
- JavaScript
- Java
- C/C++

### Career Entry Difficulty

Visualize respondents' experience breaking into the data field through segmented difficulty ratings.

### Satisfaction Analysis

Measure employee sentiment through:

- Work-Life Balance Score
- Salary Satisfaction Score

## Key Insights

### Python Leads the Industry

Python emerged as the most frequently selected programming language among survey respondents.

### Data Scientists Earn Higher Salaries

Data Scientist and Data Engineering roles generally report higher salary levels than other data positions.

### Work-Life Balance Is Strong

Survey participants reported positive work-life balance scores overall.

### Career Entry Remains Challenging

Many respondents indicated that breaking into the data industry ranged from difficult to very difficult, emphasizing the competitive nature of the field.

### Global Representation

Responses were collected from professionals across multiple countries, providing a broad perspective on careers in data.
